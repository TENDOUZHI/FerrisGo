FerrisGo的基础组件 #

视图容器 #

view #

view组件是FerrisGo最基础的组件之一，它是构成项目最常见也应用最多的基础组件

是高阶组件的基石

WARNING

view组件不支持输入文本内容

swiper #

拖动swiper组件可以生成一个轮播图在您的FerrisGo界面，这是一个高阶组件

你可以通过设置swiper组件的属性来自定义样式包括自动播放，播放延迟，轮播点，滚动条以及设置轮播项目(暂只支持图片)

WARNING

swiper组件暂时只支持一个页面部署一个

image #

image组件支持你在页面上创建一个图片，在工作区域选中组件就可以在右侧设置image组件的图片内容

基础内容 #

text #

text组件是专门用来输入文本内容的组件

这样做的目的主要是保持低耦合的组件设计

icon #

icon组件是用来设置提示消息图片组件，内置正确，警告，提醒等消息
